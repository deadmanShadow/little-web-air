# Airbnb Assistants - Property Management Landing Page

A professional, high-performance landing page for Airbnb property management services, built with a focus on modern UI/UX principles, responsiveness, and smooth animations.

## Project Overview

This project is a pixel-perfect recreation of a modern property management landing page. It features a comprehensive suite of sections including a dynamic hero area, service showcases, tiered pricing, integrated tools, and a customer testimonial marquee.

## Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Components**: [React Fast Marquee](https://www.react-fast-marquee.com/) for smooth scrolling sections

## Key Features

- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Interactive UI**:
  - Smooth scrolling marquees for "Trusted By" and "Testimonials" sections.
  - Dynamic billing toggle (Monthly/Yearly) in the Pricing section.
  - Animated floating brand logos in the Hero section.
  - Mobile-friendly navigation with a slide-out menu.
- **Custom Branding**: Integrated high-quality PNG/SVG assets for all service icons, step numbers, and industry logos.
- **Modern FAQ**: Accessible accordion-style FAQ section.

##  Project Structure

- `src/app`: Next.js App Router pages and global styles.
- `src/components`: Reusable UI components (Hero, Navbar, Pricing, etc.).
- `public`: Static assets including brand logos, service illustrations, and icons.
